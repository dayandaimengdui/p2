# p2



## 简介

西师宿管系统

软件介绍
使用SSM框架搭建后台

前端采用BootStrap框架

数据库使用MySQL

## 安装

1.数据库连接名：root 密码：root 端口：3306

2.数据库字符集：utf8 排序规则：utf8_general_ci

3.下载项目到本地，用IDEA导入项目，搭建TomCat7环境，JDK1.8，启动MySQL，设置本地端口（我的端口是8081）
